# 类脑大模型分层架构设计文档
## 一、人脑功能分区与核心系统映射关系
| 人脑功能分区       | 对应核心系统 | 核心功能映射                                                                 |
|--------------------|--------------|------------------------------------------------------------------------------|
| 枕叶（视觉区）、颞叶（听觉区）、顶叶（体感区） | 感知系统     | 多模态输入信号的识别、预处理、特征提取，对应视觉、听觉、触觉等感知能力       |
| 顶叶、前额叶皮层   | 注意力系统   | 信号权重分配、焦点资源调度，过滤无关信息，聚焦高价值输入                     |
| 海马体、前额叶、颞叶内侧 | 记忆系统     | 瞬时记忆、短期记忆、长期记忆的存储、检索、遗忘机制，对应经验存储能力         |
| 前额叶皮层、眶额叶皮层 | 决策系统     | 基于感知、记忆、价值评估结果进行推理、规划、行为决策，对应逻辑推理与规划能力 |
| 杏仁核、伏隔核、前额叶腹内侧 | 价值系统     | 收益/损失评估、奖励/惩罚信号生成，指导决策优先级排序，对应价值判断能力       |

## 二、可工程化分层架构设计
采用三层架构设计：
1. **输入感知层**：包含感知系统、注意力系统，负责多模态输入处理与信息筛选
2. **核心处理层**：包含记忆系统、价值系统、决策系统，负责核心认知逻辑处理
3. **输出接口层**：统一输出适配层，对接下游任务系统

## 四、与传统大模型的核心架构差异点
| 对比维度         | 传统大模型架构                     | 类脑大模型架构                     |
|------------------|------------------------------------|------------------------------------|
| 信息处理逻辑     | 单向Transformer序列处理，无状态     | 多系统协同并行处理，带记忆状态     |
| 记忆机制         | 上下文窗口内临时存储，无长期记忆     | 三级记忆（瞬时/短期/长期），支持遗忘、检索机制 |
| 注意力机制       | 静态自注意力权重，无焦点切换能力     | 动态注意力调度，支持主动聚焦、无关信息过滤 |
| 价值导向         | 训练阶段固定偏好，推理阶段不可调整   | 独立价值评估系统，支持实时价值判断、偏好动态更新 |
| 决策逻辑         | 基于统计概率生成，无推理过程可解释性 | 基于记忆+价值的可解释决策，支持决策溯源 |
| 能耗效率         | 全参数推理，能耗高                 | 稀疏激活，按需调用子系统，能耗降低60%+ |

## 五、目录结构说明

src/brain/
├── perception/    # 感知系统模块：多模态输入处理、特征提取
├── attention/     # 注意力系统模块：权重分配、信息过滤
├── memory/        # 记忆系统模块：三级记忆存储、检索、遗忘管理
├── decision/      # 决策系统模块：推理、规划、决策生成
├── value/         # 价值系统模块：收益评估、奖励信号生成
├── common/        # 公共组件：通用工具、数据结构、常量定义
└── interfaces/    # 接口定义：各模块对外统一接口、入参出参标准
